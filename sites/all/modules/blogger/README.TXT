Blogger
-------
This module used for show list of bloggers.


FEATURES
--------
1. Limit how many bloggers displayed
2. Whether show blogs count or not, i.e: Samuel (7), Linda (5), etc.
3. A "read more link" to show all bloggers in page format. Support pagination.
4. Case format: small, upper and wordcase.
5. Order by: none, name or blogs count
6. Support avatar. Avatar will displayed on block and page.
7. You can set avatar size: original or certain width x height. Able to keep picture aspect ratio.


INSTALL
-------
Same as other Drupal modules.
1. Extract to sites/all/modules/blogger
2. In Administer > Site Building > Modules 
   a. Activate Blog if it is not yet activated.
   b. Activate Blogger.


TO CHANGE BLOGGER SETTING
-------------------------
In Administer > Site Configuration > Blogger
- Adjust the parameters as you desire.


TO MAKE THE LIST SHOW UP IN A SIDEBAR
-------------------------------------
In Administer > Site Building > Blocks
- Find "Bloggers" in the list.
- Set the location to Left sidebar or Right sidebar.


TO MAKE THE LIST SHOW UP ONLY IN THE BLOG PAGES
-----------------------------------------------
In Administer > Site Configuration > Blogger
- Turn on "Show Blogger module when content type is blog only." so it shows up only in blog pages.


CHANGE BLOGGER LAYOUT
---------------------
Blogger.css contains css style to format Blogger layout.
